Help

Treasure Hunt is a location-based quiz game with multiple levels of difficulty.


Each stop can have different types of questions:


Multiple choice questions

Text answer questions

QR code scanner questions

Location check-in tasks (go to a place and confirm you're there)


You can skip some questions if you get stuck, but you'll earn more points by answering correctly.


Your score is tracked throughout the game and you can view the leaderboard to see how you compare with other players.
